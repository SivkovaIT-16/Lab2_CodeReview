# Список исправленных ошибок в соответствии со стандартом Epic Games C++ Coding Style
## Вариант 1
### Task 1:
1. Глобальные директивы
   - Убрано using namespace std в глобальной области видимости
   - Добавлен префикс std:: ко всем объектам стандартной библиотеки
2. Структура проекта
   - Программа разделена на модули согласно стандарту Epic Games
   - Создан заголовочный файл Header.h с прототипами функций и классами
   - Реализация методов вынесена в отдельный файл Source.cpp
   - В основном файле Task1.cpp оставлена только функция main()
3. Инкапсуляция
   - Поля класса Node сделаны приватными
   - Добавлены геттеры/сеттеры: GetData(), GetNext(), SetNext()
4. Именование классов
   - Классы переименованы в PascalCase (с заглавной буквы)
   - Пример: Node, Stack
5. Именование методов
   - Методы переименованы в PascalCase (с заглавной буквы)
   - Пример: push → Push, pop → Pop, print → Print, getTop → GetTop, clearStack → ClearStack
6. Именование параметров функций
   - Параметры переименованы в PascalCase (с заглавной буквы)
   - Имена параметров сделаны информативными
   - Пример: value → Value, stack → StackRef, D → D
7. Именование локальных переменных
   - Все локальные переменные переименованы в PascalCase (с заглавной буквы)
   - Пример: newNode → NewNode, temp → Temp, current → Current
8. Инициализация переменных
   - Все переменные инициализированы при объявлении (присвоены начальные значения)
   - Пример: int Choice = 0;, Stack MainStack;
9. Форматирование кода
   - Фигурные скобки открываются в новой строке
10. Константность методов
    - Добавлен const к методам, не изменяющим объект:
    - Пример: Print() const, GetTop() const, IsEmpty() const
11. Управление памятью
    - Добавлен деструктор ~Stack() для освобождения памяти. Деструктор вызывает ClearStack() для удаления всех узлов
12. Проверка ввода
    - Добавлена функция ClearInput() для очистки потока ввода при ошибках
    - Добавлена функция GetValidNumber() для проверки корректности вводимых данных
    - Добавлена обработка некорректного ввода (буквы вместо цифр)
13. Локализация
    - Добавлена поддержка русского языка: setlocale(LC_ALL, "Russian");
14. Комментарии
    - Комментарии начинаются с заглавной буквы
    - Комментарии заканчиваются точкой
